# Crear un Ticket

## Objetivo
Registrar una nueva solicitud de soporte en el sistema.

## Pasos

### 1. Acceder al formulario
Navega a **Tickets > Crear Ticket** desde el menú principal.

### 2. Completar información básica

#### Asunto
Escribe un título claro y conciso que describa el problema.

**Ejemplo**: "Error al iniciar sesión en el sistema"

#### Descripción
Proporciona detalles completos del problema:
- ¿Qué estabas haciendo cuando ocurrió?
- ¿Qué mensaje de error aparece?
- ¿Cuándo comenzó el problema?

:::tip Consejo
Usa el editor de texto enriquecido para formatear tu descripción con negritas, listas y enlaces.
:::

### 3. Seleccionar categorización

#### Departamento
Selecciona el departamento relacionado con tu solicitud.

#### Categoría
Elige la categoría que mejor describe tu problema. Las opciones cambian según el departamento seleccionado.

#### Subcategoría
Selecciona la subcategoría específica. Esto ayuda a asignar el ticket al equipo correcto.

:::info Nota
La prioridad se establece automáticamente según la subcategoría, pero puedes cambiarla si es necesario.
:::

### 4. Establecer prioridad

Selecciona la urgencia del ticket:
- **Alta**: Problemas críticos que bloquean el trabajo
- **Media**: Problemas importantes pero no urgentes
- **Baja**: Consultas o mejoras menores

### 5. Adjuntar archivos (opcional)

Si tienes capturas de pantalla o documentos relevantes:
1. Haz clic en "Seleccionar archivos"
2. Elige los archivos desde tu computadora
3. Verifica que se hayan cargado correctamente

:::warning Límites
- Tamaño máximo por archivo: 10MB
- Formatos permitidos: PDF, PNG, JPG, DOCX
:::

### 6. Enviar el ticket

Haz clic en **"Enviar Ticket"** para registrar tu solicitud.

## Resultado Esperado

Verás un mensaje de confirmación indicando que el ticket fue creado exitosamente. El sistema:
- Asigna automáticamente un número de ticket
- Inicia el flujo de trabajo correspondiente
- Notifica al equipo responsable

Puedes ver el estado de tu ticket en la lista de tickets.

## Errores Comunes

### "Debe seleccionar una categoría"
**Causa**: No seleccionaste todos los campos obligatorios.
**Solución**: Completa Departamento, Categoría y Subcategoría.

### "Error al cargar el archivo"
**Causa**: El archivo es muy grande o tiene un formato no permitido.
**Solución**: Verifica el tamaño y formato del archivo.
