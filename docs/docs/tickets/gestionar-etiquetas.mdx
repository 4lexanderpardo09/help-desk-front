# Gestión de Etiquetas

## Objetivo
Organizar y clasificar tus tickets mediante etiquetas personalizadas (Tags).

## Características
- **Privacidad**: Las etiquetas son personales. Solo tú puedes verlas y gestionarlas.
- **Visualización**: Aparecen como distintivos de colores en la lista de tickets y en la vista de detalle.
- **Filtrado**: Te permiten identificar rápidamente grupos de tickets de interés.

## Crear y Asignar Etiquetas

### Desde el Listado de Tickets
1. Ubica la columna **Etiquetas** en la tabla de tickets.
2. Haz clic en el botón `+` (Agregar etiqueta) o en las etiquetas existentes de una fila.
3. Se abrirá el **Gestor de Etiquetas**.
4. **Crear nueva**:
   - Escribe el nombre de la nueva etiqueta en el buscador.
   - Selecciona un color distintivo.
   - Haz clic en **Crear**.
5. **Asignar existente**:
   - Haz clic sobre cualquier etiqueta de tu lista para asignarla o removerla del ticket actual.
   - Las etiquetas seleccionadas aparecerán marcadas.

### Desde el Detalle del Ticket
1. Ingresa al ticket.
2. En la barra lateral o superior, busca la sección de etiquetas.
3. El proceso es idéntico al del listado.

## Editar o Eliminar Etiquetas
Dentro del **Gestor de Etiquetas**:
- **Editar**: Haz clic en el icono de lápiz para cambiar el nombre o color.
- **Eliminar**: Haz clic en el icono de papelera para borrar la etiqueta definitivamente.
  > **Nota**: Al borrar una etiqueta, esta desaparecerá de todos los tickets a los que estaba asignada.

## Límites
- Puedes crear tantas etiquetas como necesites.
- Recomendamos usar colores variados para facilitar la distinción visual.
