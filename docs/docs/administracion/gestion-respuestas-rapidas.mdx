# Gestión de Respuestas Rápidas

El módulo de **Respuestas Rápidas** permite gestionar un banco de respuestas predefinidas o códigos de error comunes que facilitan la gestión de tickets.

## Acceso

Para acceder, seleccione **Administración > Respuestas Rápidas** en el menú lateral.

## Listado de Respuestas

- **Título**: Nombre corto o código de la respuesta.
- **Tipo**: Clasificación (Técnico, Informativo, Administrativo).
- **Estado**: Activo/Inactivo.

## Crear Nueva Respuesta

1.  Haga clic en **Nueva Respuesta**.
2.  **Título**: Ingrese el título o asunto (ej: "Error 404").
3.  **Descripción**: Redacte el contenido detallado de la solución o procedimiento.
4.  **Tipo**: Seleccione la categoría correspondiente.
5.  **Estado**: Defina si estará disponible para su uso.
6.  Haga clic en **Guardar**.

## Uso

Estas respuestas pueden ser utilizadas posteriormente por los agentes al atender tickets para insertar soluciones estandarizadas rápidamente.
