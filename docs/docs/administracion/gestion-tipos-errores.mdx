# Gestión de Tipos de Error

El módulo de **Tipos de Error** permite catalogar los errores estándar reportados en el sistema para facilitar su clasificación y análisis.

## Acceso

Para acceder, seleccione **Administración > Tipos de Error** en el menú lateral.

## Listado de Tipos de Error

- **Nombre**: Nombre del error.
- **Descripción**: Descripción técnica o funcional.
- **Estado**: Activo/Inactivo.

## Crear Tipo de Error

1.  Haga clic en **Nuevo Tipo de Error**.
2.  Ingrese el **Nombre** del error y su **Descripción**.
3.  Seleccione el **Estado** (Activo).
4.  Guarde.

:::info Clasificación
Estos tipos de error se utilizan al categorizar tickets cerrados o en proceso, permitiendo generar reportes de incidencias recurrentes.
:::
