# Gestión de Perfiles

El módulo de **Perfiles** administra los perfiles de usuario que agrupan permisos y roles específicos dentro del sistema.

## Acceso

Para acceder, seleccione **Administración > Perfiles** en el menú lateral.

## Listado de Perfiles

- **Nombre**: Nombre descriptivo del perfil.
- **Descripción**: Detalle del propósito del perfil.
- **Estado**: Activo/Inactivo.

## Crear Perfil

1.  Haga clic en **Nuevo Perfil**.
2.  Ingrese el **Nombre** y una **Descripción** opcional.
3.  El estado será **Activo** por defecto.
4.  Guarde.

:::tip Uso
Los perfiles se utilizan para asignar permisos masivos a usuarios o para definir reglas de mapeo en la creación de tickets.
:::
